IPL Match Analysis Dashboard(2008-24)
Project Overview
This Power BI dashboard provides a comprehensive analysis of IPL matches from 2008 to 2024. The goal of this project is to visualize team performances, toss impacts, and key match statistics in a professional and interactive way.

Key Features
Team Performance Leaderboard: Tracks total wins for all franchises with data cleaning (merging team name changes).

Toss Decision:there are i used donut chart to represent of Toss Decision by Team,here we showed how many times won the toss what would be they choose(Bat,field)

Win Strategy Analysis: A deep dive into "Batting First" vs "Chasing" success rates using Donut charts.

Match Intensity: Categorizes matches into "Close Finishes", "Standard", and "One-sided" based on victory margins.

Toss Factor: Analyzes if winning the toss actually helps in winning the match.

Top 10 Highest Targets: A dynamic table showing the highest scores set in IPL history.

Top 10 Player of matches:In this dashboard i visualized top 10highest player of match player by owen performance.

Total super over by Total matches: here we showes total super over happens in the entire total season of ipl.

Count of Total Matches played by Season:In this kpi i showed the count of matches played in a specific season, with the help of season slicer we can see the data in this chart.

Total Wins &venue by Winner: In this visualization we can see here winner total wins in the specific venue and representing winning count how many matches won the this venue

Total Matches by City:In this chart i represent the total matches played in which city and we show the data with the help of data level (number of mathces)

Tools & Technologies Used
Power BI Desktop: For visualization and dashboarding.

Power Query: Used for data cleaning (handling 'NA' values, renaming teams, and formatting).

DAX (Data Analysis Expressions): Created custom measures for Total Wins, Win %,Toal matches ,average target runs and Match Categories

How to Use

Download the .pbix file from this repository.

Open it in Power BI Desktop.

Use the Navigation Buttons to switch between Page 1 (Ipl Analysis) and Page 2 (Player&venue Insight).

Use Slicers to filter data by Season or City.
